Obten empleo fácil

Un sitio web especializado en demostrar las habilidades y beneficios de contratar a un empleado neurodivergente otorga a las organizaciones un aumento de la productividad y mejor búsqueda de empleabilidad en roles que el neurodivergente puede aportar a la empresa. ALCANCES Y LÍMITES Alcances La plataforma web permitirá el registro tanto de usuarios neurodivergentes como de los encargados de publicar en el sistema las ofertas de trabajo de las empresas. El sistema web permitirá que los empleadores generen sus propios formularios de postulación para los trabajos que están ofertando. El sistema web permitirá que los usuarios neurodivergentes puedan presentar su currículum vitae al realizar su registro en la plataforma. El sistema web permitirá que los usuarios neurodivergentes puedan colocar sus capacidades y cualidades al momento de llenar el formulario de postulación. La plataforma web contará con un sistema de notificaciones interno para que el usuario neurodivergente pueda ver si se le aceptó o rechazó la postulación de trabajo. El sistema web permitirá que el usuario que oferta el trabajo, al aceptar una solicitud de postulación, pueda programar una entrevista con el postulante. El sistema web permitirá que el usuario postulante pueda ver las entrevistas programadas para poder confirmar su asistencia. La plataforma web contará con un apartado de ayuda donde el usuario neurodivergente podrá aprender a utilizar la plataforma fácilmente. Límites El sistema web solamente ofertará propuestas de empleo en Bolivia. La página web no permitirá que el usuario neurodivergente realice modificaciones en un formulario de postulación ya enviado. El sistema web solamente contará con una base de datos local. El sistema web será netamente un sistema de información transaccional. Las notificaciones de la plataforma no podrán ser vistas de manera externa al inicio de sesión de la página. La plataforma web no contará con una versión móvil, lo que limita el acceso a la página a través de dispositivos móviles.
